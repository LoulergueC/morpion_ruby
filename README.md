# Morpion en Ruby

Personnalisable pour deux joueurs !
Définissez vos noms et vos propres symboles pour vos parties.

Le score final sera affiché !

# Deployment

1. Clone the repo
   run `git clone https://github.com/LoulergueC/morpion_ruby.git`

2. Install gems
   run `bundle install`

3. Launch the game
   run `ruby app.rb`
